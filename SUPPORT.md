# Support

Use the right channel so maintainers can respond efficiently.

## Usage Questions

For setup help, output interpretation, integration questions, or general discussion, use GitHub Discussions:

- https://github.com/callchain/callchain/discussions

## Bug Reports

For defects or regressions, open a GitHub issue with the bug report template and include:

- the CallChain version
- the target language(s)
- the command you ran
- a minimal reproduction when possible

## Security Reports

Do not open public issues for security-sensitive reports. Follow the guidance in [SECURITY.md](SECURITY.md).

## Contributions

If you already know the change you want to make, open a pull request and follow [CONTRIBUTING.md](CONTRIBUTING.md).
