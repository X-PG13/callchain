# Code Of Conduct

## Our Pledge

We want CallChain to be a professional, respectful project for contributors and users. Participation in this project means engaging constructively and treating other people with respect.

## Expected Behavior

- Be respectful in technical disagreement.
- Focus on code, evidence, and user impact.
- Assume good intent, but be explicit about risks and tradeoffs.
- Accept feedback and course-correct quickly when needed.

## Unacceptable Behavior

- Harassment, abuse, threats, or discriminatory language.
- Repeated bad-faith argument or deliberate disruption.
- Publishing private information without permission.
- Trolling, personal attacks, or hostile escalation.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate this code of conduct.

## Reporting

Report conduct issues to the maintainers through the repository issue tracker or the security/reporting channel documented in `SECURITY.md`.
